The brain — luminous concept atlas

open file open →

A 3D, bioluminescent map of the model's interior: ~1000 real SAE features (Qwen2.5-7B) clustering by decoder geometry into 26 emergent lobes. Type a prompt and it pulls the genuinely-relevant concepts live from the full 131k-feature space — named by Neuronpedia — that bloom as labelled lights beside their neighbours, while it answers you in a remembered conversation. Nothing decorative; every light is a feature that actually fired.

clozn .venv python research/brain_server_7b.py 8090 # the 7B brain + full-space readout

Watch it remember you

open file open →

A frozen local 4-bit Qwen-7B holds a 20-turn conversation, is never told a single rule, then distils what it noticed about you into a 16-vector memory by test-time training. On prompts it never saw — the conversation gone from its context — it answers like it knows you (hard-magic fantasy, a vegetarian dinner, kept short). Real model, real TTT. Press play and watch the memory form across the conversation.

cloze .venv python research/memory_timeline.py # generate the real run clozn .venv-sae python research/wire_memory.py # wire it into memory.html

Watch it denoise — diffusion

open file open →

A real Dream-7B diffusion LM fills a board of masked slots in parallel over 20 passes — the answer crystallising out of the mask, out of order (it commits "Jupiter" before "the solar system", function words last), each token lit by its confidence. The dLLM counterpart to watch-it-think: for a diffusion model the thinking is literally visible.

PYTHONPATH=engine/lab cloze .venv python research/denoise_server.py --port 8082 # LIVE: type your own prompts PYTHONPATH=engine/lab cloze .venv python research/denoise_capture.py # or capture a static run
