Clozn — a local runtime where the model's interior is legible and editable.
Nothing faked. A frozen GPT-2-small (124M), never fine-tuned — one model behind this whole page. The memory is a literal list of entries injected by one forward hook at a single mid-layer; each value is the answer token's unembedding direction, which decodes to that word by construction. A multi-token name uses the answer's first token's direction — recall then promotes that first token (the card and the fired line say so; the later pieces of the name are not separately addressed). Addressing is gated top-1 cosine: a query fires the nearest stored key only if it clears the match-strictness gate — otherwise no vector is injected and with memory equals baseline exactly ("stayed silent"). That gate is the slider above and is fully live: loosen it and softer paraphrases start firing (and false fires creep in); the real nearest cosine is shown on every ask so you see exactly where each query sits. Every probability shown is the model's real output.

The "what it's thinking" panel is a read, not a decision. Each named concept is a diff-in-means direction (mean residual over positive example sentences minus contrast sentences) built once at startup on this same frozen model. As it reads your prompt we project the residual at every token onto each direction; the honesty knob is the null: a raw projection is meaningless because bigger residuals project more onto everything, so at each token we also project onto hundreds of random directions of equal norm and report z = how many σ the real direction sits above that random band. A concept lights up only where it clears the null — what's present in its state, not what it "decided" or will say. The same backend powers this page; all requests are same-origin.
