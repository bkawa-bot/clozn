Clozn — a local runtime where the model's interior is legible and editable.
Nothing faked. A frozen GPT-2-small (124M), never fine-tuned. The memory is a literal list of entries injected by one forward hook at a single mid-layer; each value is the answer token's unembedding direction, which decodes to that word by construction. Addressing is gated top-1 cosine: a query fires the nearest stored key only if it clears the gate — otherwise no vector is injected and with memory equals baseline exactly ("stayed silent"). Every probability shown is the model's real output. The same backend powers this page; all requests are same-origin.
